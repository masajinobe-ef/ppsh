## Installation

1. Clone the repository: `git clone https://github.com/masajinobe-ef/ppsh`
2. Go to the directory: `cd ppsh`
3. Launch PPShell: `./ppsh.ps1`

## Using

```bash
./ppsh.ps1 # Count = 2
./ppsh.ps1 -Count 10
./ppsh.ps1 -SetFastestDns
./ppsh.ps1 -SetDnsServers ("8.8.8.8","8.8.4.4")
```
